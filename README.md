# Quartile-Based Clinical Scoring (QBCS) & Voting Feature Selection

This repository contains the implementation of the QBCS algorithm, a clinical scoring system for analyzing liver biomarkers (ILPD Dataset). All development was performed via Google Colab to ensure portability and ease of execution.
More information in the paper "Liver disease diagnosis: fuzzy c-means and scoring ranking approach", under revision (05/02/26)
